Handbuch zur Verwendung von MuViBee

1. Vorwort

Der Name MuViBee stammt aus der umgangssprachlichen Abkürzung von Music Video Bibliothek. Das fleißige Bienchen einem beim Finden von Büchern, Musik und Videos.

Ziel der Software ist es, jeder Altersklasse eine einfache und intuitive Medienverwaltung zu ermöglichen, ohne große PC Kenntnisse vorauszusetzen.

 

2. Programmübersicht

Die Benutzerobefläche ist in fünf Bereiche gegliedert.

  1. das Suchfeld
  2. die Kategorie Tabs
  3. das Hauptfenster
  4. die Statusleite
  5. zusätzliche Optionen

 

2.1. das Suchfeld

Geben Sie einfach einen Begriff im Suchfeld ein und klicken Sie anschließend auf "Suchen". Das fleißige Bienchen wird Ihnen umgehend alle Ergebnisse im Hauptfenster anzeigen. Sollte Ihre Suche ohne Ergebnis oder nicht passend genug ausfallen, klicken Sie mit der linken Maustaste auf das Feld "Erweiterte Suche".[zurück zum Seitenanfang]

 

2.2. die Kategorie Tabs


Um die Software stets übersichtlich zu halten, ist das Hauptfenster in verschiedene Tabs / Reiter gegliedert, wie man es auch von Karteikarten kennt. Im Papierkorb haben Sie die Möglichkeit ausversehen gelöschte Elemente wiederherzustellen. [zurück zum Seitenanfang]

 

2.3. das Hauptfenster

Bei der Verwendung von MuViBee arbeiten Sie hauptsächlich im Hauptfenster. Hier erhalten Sie auf der Startseite eine Übersicht all Ihrer Medien und eine Übersicht überfälliger Medien (siehe mehr dazu unter Verleihfunktion).
Nach Auswahl eines Tabs, erhlten Sie immer im oberen die Möglichkeit nach verschiedenen Eigenschaften zu sortieren.
Im unteren Bereich finden Sie den Button "Hinzufügen". Je nachdem in welchem Tab Sie sich nun befinden, können Sie ein neues Medium hinzufügen. Nach dem Klick auf "Hinzufügen" können Sie nun auswählen, ob Sie das Medium mit Hilfe der EAN / ISBN anlegen wollen oder manuell. Bei der Verwendung von EAN / ISBN, benötigen Sie jedoch eine aktive Internetverbindung. [zurück zum Seitenanfang]

 

2.4. die Statusleiste

In der Statusleiste erhalten Sie stest alle Meldungen, wie "erfolgreich gespeichert", oder eine Fehlermeldung. [zurück zum Seitenanfang]

 

2.5. zusätzliche Optionen

Hier können Sie zum Einen die Sprache Ihrer Benutzeroberfläche ändern, diese Hilfe aufrufen oder Informationen zum aktuellen Softwarestand abrufen. [zurück zum Seitenanfang]

 

 

3. Funktionen

  1. Medium anlegen
    1. per ISBN / EAN
    2. manuell
  2. Erweiterte Suche
  3. Wiederherstellen gelöschter Elemente
  4. Verleihfunktion

 

3.1 Medium anlegen

3.1.1. per EAN / ISBN anlegen

In sofern Sie über eine aktive Internetverbindung verfügen, können Sie mit Hilfe der auf Büchern befindlichen ISBN Nummer, bzw. der auf allen Medien befindlichen EAN Nummer, bequem per Internet alle Informationen zu Ihrem Medium anlegen. Das fleißige Bienchen fügt Ihnen dann alle Informationen zum Produkt hinzu. Anschließend können Sie einzelne Angaben bearbeiten. Sie sparen sich dadurch aber das manuelle Anlegen und viel Zeit. [zurück zum Seitenanfang]

3.1.2. manuell

Sollte das automatische Anlegen fehlschlagen oder nicht möglich sein, haben Sie weiterhin die Möglichkeit Ihr Medium manuell anzulegen. Nach der Auswahl "manuell" öffnet sich im rechten Bereich des Hauptfensters ein Formular um Ihr Medium von Hand anzulegen. [zurück zum Seitenanfang]

 

3.2. die erweiterte Suche

Mit Hilfe der erweiterten Suche ist es möglich, gezielt nach einzelnen Eigenschaften zu suchen. Beispielsweise nur Videos, Musik oder Bücher.
Darüber hinaus verfügt die erweiterte Suche über eine so genannte Volltextsuche. Das bedeutet, dass Sie nie den vollständigen Titel eingeben müssen. Angenommen Sie haben die Filme " P.S. Ich liebe Dich " und " Liebe braucht keine Ferien" in Ihrem Datenbestand. Wenn Sie nun im Titel einfach "Liebe" eingeben, da Ihnen der vollständige Name nicht mehr bekannt ist, werden Ihnen nun diese beiden Filme als Ergebenis angezeigt. [zurück zum Seitenanfang]

 

3.3. Wiederherstellen gelöschter Elemente

Im Papierkorb können Sie zuvor gelöschte Medien wiederherstellen oder vollständig löschen. [zurück zum Seitenanfang]

 

4.4 Verleihfunktion

Mit Hifle der Verleihfunktion behalten Sie stets ein Übersicht über die Medien, die Sie Ihren Freunden und Bekannten ausgeliehen haben. Somit vergessen Sie nie ein Buch zurückzufordern. Auf der Übersichtsseite können Sie jederzeit sehen, wieviele Bücher / Videos / CDs Sie aktuelle verliehen haben. Darüber hinaus können Sie ein Enddatum setzen. Das fleißige Bienchen erinnert Sie dann automatisch, das Buch zurückzufordern. Überfällige Medien werden Ihnen in der Übericht angezeigt. [zurück zum Seitenanfang]
