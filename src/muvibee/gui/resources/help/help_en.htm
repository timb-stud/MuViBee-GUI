MuViBee Guide

1. Foreword

The name MuViBee comes from the colloquial abbreviation for Music Video Library. The hard-working bee helps you finding your books, music and video.

The aim of the software is to enable all persons in simple and intuitive media management, with little computer skills.

 

2. Overview

The surface is divided into five sections.

  1. the search box
  2. the Tabs
  3. the main window
  4. the status bar
  5. more options

 

2.1. the search box

Simply enter a keyword in the search box and then click the Button "Search". The hard-working bee will give you instant results in the main window. If your search results will fail or will not fit enough for you, you can click onto "Advanced Search".[back to top]

 

2.2. the tabs


To keep the software open, the main window is devided in different tabs, as known from index cards. In the trash you can restore accidentally deleted items. [back to top]

 

2.3. the main window

When using MuViBee you work primarily in the main window. Here you get an overview of all your media and overdue media (see more at rental function).
After selecting a tab, you will always find the possibility to sort for different properties.
In the lower section you will find the button "Add". Depending on which tab you are now, you can add a new medium. After clicking the Add button, you can choose whether you want to create the medium with the EAN / ISBN or manually. The use of EAN / ISBN, you will need an active Internet connection. [back to top]

 

2.4. die status bar

In the status bar, you will receive all messages, such as "successfully saved", or an error message. [back to top]

 

2.5. more options

Here you can change the language of your user interface, open this help dialog or retrieve information about the current software version. [back to top]

 

 

3. Functions

  1. Insert media
    1. ISBN / EAN
    2. manuell
  2. Advanced Search
  3. restore accidentally deleted items
  4. rental function

 

3.1 Insert media

3.1.1. EAN / ISBN

Unless you have an active Internet connection, you can use the ISBN number located on books, or create the media with theEAN number. The hard-working bee will add all the information to this product. After taht you can edit individual data. This will avoid the manual creation and a lot of time. [back to top]

3.1.2. manuell

If the automatic creation will fail you have still the opportunity to create the media manually. After selecting "manually" it opens at the right side of the main window a form to get the media inserted. [back to top]

 

3.2. the advanced search

Using the advanced search, it is possible to search for individual properties. For example, only videos, music or books.
In addition, the advanced search is based on a so-called full-text search. This means that you never have to type the full title. Suppose the movies "PS I love you" and "Love does not need vacation" in your database and you enter simply "love", both movies will appear. [back to top]

 

3.3. Restore accidentally deleted items

In the trash you can recover deleted media or completely remove from your database. [back to top]

 

4.4 Rental function

With the distribution function you can always keep an overview of the media you have lent to friends and acquaintances. So you never forget recover a book. On the summary page, you can always see how many books / Videos / CDs have been rent at present. You can also set an end date. The hard-working bee will remind you automatically to recover the book. Overdue media is displayed on the overview. back to top]
